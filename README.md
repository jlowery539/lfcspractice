These are sets of interactive html files that provide lab exercises for practing for the LFCS exam. Some of the labs may not be 100 percent accurate, but it's something to learn from and build troubleshooting skills. If you find commands and steps in the labs that aren't working correctly, use the MAN pages in linux and web resources to troubleshoot the commands to make them work correctly. You'll want to use virtual machines to run these labs. To use them, just download the HTML files to a directory and open them using any browser you'd like. I used AI to generate these because I couldn't find a ton of hands on resources out there to practice these topics.
